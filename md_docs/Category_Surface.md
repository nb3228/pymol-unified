# Details

There are a few modes for displaying surface cavities. Normal (0), inverted (1) and more. 

# Usage
    
    
    # set the mode to (integer)
    set surface_cavity_mode, (integer)
    

# Examples
    
    
    # set the mode to inverted
    set surface_cavity_mode, 1
